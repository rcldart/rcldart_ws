# rcl_interfaces

Generated Dart bindings for the ROS 2 `rcl_interfaces` interface package
(messages/services), part of the
[rcldart](https://github.com/rcldart/rcldart) workspace. Produced by
`rosidl_generator_dart` from the package's `.msg`/`.srv` definitions so the types
can be used from Dart/Flutter.

These are typed Dart classes mirroring the ROS 2 package. They are **generated,
not hand-edited** — regenerate from the interface package rather than changing
`lib/` directly. Bytes on the wire are handled by
[`ros2_cdr`](../ros2_cdr); transport by `dds_direct` / `zenoh_ros2`.
